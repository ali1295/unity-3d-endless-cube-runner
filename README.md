# 🎮 Unity 3D Endless Cube Runner

A simple **3D Endless Runner game** built using **Unity Engine** and **C#**.  
The player controls a cube that must avoid obstacles and survive as long as possible while the speed gradually increases.

---

## 📌 Features

- 🟦 Smooth cube movement system  
- 🚧 Random obstacle spawning  
- 🏃 Endless runner gameplay loop  
- ⚡ Increasing difficulty over time  
- 🎮 Simple keyboard controls  
- 💥 Collision-based game over system  
- 🎨 Basic 3D environment using Unity  

---

## 🛠️ Built With

- Unity Game Engine  
- C# (Scripting)  
- Unity Physics System  
- Visual Studio / Rider  

---

## 🎮 Gameplay

- The player controls a **cube character**
- Avoid incoming obstacles
- Survive as long as possible
- Score increases with time/distance

---

## 🎮 Controls

| Action | Key |
|--------|-----|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Restart | R |
